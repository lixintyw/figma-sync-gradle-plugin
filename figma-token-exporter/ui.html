Token Binding Exporter

Select COMPONENT / COMPONENT_SET / INSTANCE nodes in your Figma file, then click Export Bindings to extract variable-linked fill colors. The output JSON is compatible with the figma-sync-gradle-plugin.

Next step: Save the JSON output to your Android project at app/src/main/assets/figma_plugin_export.json. The Gradle plugin will read it as a fallback data source for design token sync.
